# do_type

# 사용법
```
<body>
  <h1 class="do_type" title="Hello, my name is Erik Minimis.*I do creative">title</h1>
  <p title="hello,* my name is kwon">type</p>
  
  <!-- 플러그인 -->
  <script src="./do_type.js"></script>
  
  <!-- 작성 스크립트 -->
  <script>
  
    // Initailize
    do_type.init();
    
    // run auto type
    do_type.type('element name');
  
  </script>
</body>
```
